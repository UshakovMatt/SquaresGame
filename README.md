Игра запускается, через терминал проекта
cmd
далее
.\gradlew.bat run - Запуск через Gradle
.\gradlew.bat build - Сборка проекта
.\gradlew.bat clean - Очистка
.\gradlew.bat clean build - Пересборка с очисткой

В консоли 
GAME X, user W, comp B
	GAME выбираем размер X (поля) от 3 до 20, выбор пользователь или компьютер с указанием цветов белый или черный
	также возможен comp W, comp B; user W, user B
MOVE x, y  
	Движение по полю через команду MOVE x - столбец, y - строка
HELP 
	Выдача памятки о том, как запустить игру, как делать ходы, досрочное завершение
EXIT
	Досрочное завершение игры
